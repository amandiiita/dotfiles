# 🎀 My Coquette macOS Setup 🎀

¡Hola! 🌸 Bienvenid@ a mis dotfiles. 
Este repositorio guarda la configuración de mi terminal para que se vea mucho más bonita.

### 🪞 ¿Qué incluye?
* **Prompt:** Starship personalizado con temática *girly* (🎀, ✧, 🌸).
* **Mejoras visuales:** Eza y Bat para que las listas de archivos y el código tengan colores lindos.
* **Fuente:** FiraCode Nerd Font para que todos los íconos se rendericen perfectos.
